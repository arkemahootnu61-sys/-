# 51单片机学习笔记

## 002单片机简介

- 单片机是一个芯片上的微型计算机，体积小，价格便宜，易于编程
- STC89 STC厂商 89内部代号 
  - C 工作电压 5.5V - 3.3V.    LE 3.6V - 2.0V
  - 52，8K字节代表程序空间大小，硬盘
  - RC代表RAM，电脑的内存512字节，RC+，1280字节
  - 40 工作频率 MHZ
  - I工作温度  -40 - 85工业级别    C级别  0 - 70
  - LQFP 封装类型 PDIP，性能没有差异，外形不同
  - 44，44个引脚 

## 003引脚的功能

- 电源引脚 Vcc，Gnd正负极
- RST复位引脚，重新开始执行程序，相当于重启，我们的是高电平复位
- 时钟引脚，XTAL1，XTAL2，接外部晶振，连接时钟源，提供时钟信号
- 通用输入输出引脚GPIO,一共39个，只有高低电平，
  - P0 8个
  - P1 8
  - P2 8
  - P3 8
  - P4 7
  - 除了通用输入输出，还有一些是有其他作用，叫做复用引脚，可以有很多个功能

## UART通信
- 串行通信
	- 两个设别只通过一根数据线传
	- 并行就是多条数据线 
- **单工与双工通讯**
	- 只能单向通信
	- 可以双向通信
		- 半双工，不能同时进行
		- 双工，可以同时进行
	- **同步与异步通讯**
		- 同步，可以规定下降沿，也可以规定上升沿
			- ![[Pasted image 20260715161538.png]]
		- 异步，统一发送方发送的频率与接收方接收信息的速率，不需要额外的时钟信号，内部的时钟是不同步的
- **UART 是一种异步，全双工，串行的通信接口**
	- ![[Pasted image 20260715162154.png]]
#### UART通信协议
-  数据格式
	- 数据是逐帧发送的，每个数据帧包括起始位，数据位，校验位，停止位
	- ![[Pasted image 20260715162713.png]]
	- 数据位一般是5到9位，一般是8位，校验位，校验当前帧的正确性
		- 校验：发送方与接收方通过算法来看数据是否相同
			- ![[Pasted image 20260715163407.png]]
	- 发送方与接收方约定
		- 波特率：数据的传输速率，每秒传输的基本符号数。
		- 比特率：每秒传输的比特数。
		- 在二进制当中，波特率与比特率是一样的
		- 比如说，我一定会成功，是6个比特，但是很多个字节
		- **约定好数据的位数**
		- **校验位约定有没有以及校验方法**
		- **停止位1位还是两位，都需要约定一下**
