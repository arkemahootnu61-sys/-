# 数电学习笔记

## 一，数制与码制

- 二进制信号：在电路中用高低电平来表示1，0两种信号
- 八进制，十进制，十六进制（ABCDEF）
- 按照权重展开来，得到十进制
- 十进制转化成二进制，长除法，直接靠感觉就可以，其他进制以十进制为媒介接可以
- 加法运算，逢二进一，乘法跟正常一样的，减法与除法先转化成十进制，在转化会二进制回去计算
- 源码，反码，补码
  - 无符号数与有符号数，有符号数加上首位表示符号位，0表示正数，1表述负数
  - 对于正数，源码反码补码都是相同的
  - 对于负数，符号为不变，其他位取反，得到反码，反码加一得到补码，相反过程或者在求补码，得到源码
    - 计算机的运算都是用补码来计算

#### 码制

- 8421码（BCD码），余2码，只能表示一位十进制数0-9，未列的就是无效的，余三码就是加8421加上3（0011）

- | 十进制数 (Decimal) | 4位标准二进制 (Binary) | 8421 BCD码 | 余3码 (Excess-3) |
  | ------------------ | ---------------------- | ---------- | ---------------- |
  | 0                  | 0000                   | 0000       | 0011             |
  | 1                  | 0001                   | 0001       | 0100             |
  | 2                  | 0010                   | 0010       | 0101             |
  | 3                  | 0011                   | 0011       | 0110             |
  | 4                  | 0100                   | 0100       | 0111             |
  | 5                  | 0101                   | 0101       | 1000             |
  | 6                  | 0110                   | 0110       | 1001             |
  | 7                  | 0111                   | 0111       | 1010             |
  | 8                  | 1000                   | 1000       | 1011             |
  | 9                  | 1001                   | 1001       | 1100             |

- 格雷码，从0000开始，最后一位从0110循环变化，倒数第二位是00111100，倒数第三位0000 1111 0000 1111变化，倒数第四，0000 0000 1111 1111
  - 特点，编码顺序依次变化的时候，相邻两个代码之间只有一位发生变化
- ASCII码，0表示48，65表示A，97表示a

## 二，逻辑代数基础

- 二值逻辑运算，与，或，非，三种，这个学了很多遍了，0为假，非0为真

  - 与Y = AB,AB都为真的时候才是真
  - 或，只要有一个1就为1，Y = A + B
  - 非，取反

  - 与非，与加非，在与门的基础上打了个圈，与门还是与非门都是对0敏感的电路

  - 或非，或门或者或非门都是对1敏感的电路

  - 异或，不同的时候为1，相同的时候为0，框框里面等于1

  - 同或，相同的时候是1，不同的时候为0，，框框里面只有等于号
